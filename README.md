📖 About
FUTA CBT Hub is a fully offline-capable, single-file web app built specifically for 100-level students of the Federal University of Technology, Akure (FUTA). It provides structured CBT practice across all major 100-level courses, with timed test modes, topic mastery tracking, answer review, and an AI-powered mini academic tutor.
Built by a student, for students. No subscription. No login required. Just open and practice.

📊 Smart Features
Topic Mastery Tracker — Tracks performance per topic with a weighted blend (40% history + 60% latest)
Question Navigator — Jump to any question, flag questions for review
Answer Review — Filter wrong answers, see correct answers + explanations
Test History — Full log of all completed sessions with scores and time
Exam Timetable — Live countdown to each upcoming exam with "Practice Now" shortcut
Ask the Tutor — In-app academic chatbot trained on PHY101 and MTS101 topics

🔄 How to Update
Open index.html in this repository
Click the pencil ✏️ (Edit) icon
Make your changes (add questions, fix errors, update timetable, etc.)
Click Commit changes
GitHub Pages rebuilds automatically in ~1–2 minutes
🐛 Found an Error?
If a question has the wrong answer or explanation:
Use the Report Bug section inside the app, or
Open an Issue on this repository with the question number and course
